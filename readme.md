# json-svg-icons

> My unreal module


## Install

```
$ npm install --save json-svg-icons
```


## Usage

```js
var reactSvgIcons = require('json-svg-icons');
reactSvgIcons;
//=> {
//      facebook:...,
//      twitter:...,
//}
```

or individually: 
```js
var facebook = require('json-svg-icons/dist/facebook.json');
```
